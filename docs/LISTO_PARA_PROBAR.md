## 🐛 Si Algo No Funciona

### Problema: No veo el botón 📄
```
Solución:
1. Verifica que estás en /editor (no en /login o /gallery)
2. Refresca la página (F5)
3. Verifica consola (F12) por errores
```

### Problema: El modal no se abre
```
Solución:
1. Abre consola (F12)
2. Busca errores en rojo
3. Intenta desde el chat: "Cargar documento"
4. Refresca la página
```

### Problema: No se genera el diagrama
```
Solución:
1. Verifica que procesaste el documento primero
2. Verifica que hay entidades detectadas
3. Usa los archivos de ejemplo
4. Revisa consola por errores
```
