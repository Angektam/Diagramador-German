# Shared Types

Tipos e interfaces compartidas entre frontend y backend para mantener consistencia.

## Uso

### En Frontend (Angular)
```typescript
import { Diagram, Table } from '../../shared/models/diagram.interface';
```

### En Backend (Node.js)
```typescript
import { Diagram, Table } from '../shared/models/diagram.interface';
```

## Contenido

- `models/diagram.interface.ts` - Interfaces de diagramas, tablas y relaciones
